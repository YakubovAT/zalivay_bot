# Zalivai Bot — Платформа для масштабирования контента в Pinterest

## О проекте

**Zalivai Bot** — Telegram бот для создания и массового распределения контента товаров в Pinterest.

Платформа решает главную проблему: как из одного товара создать множество контента и разместить его везде с минимумом усилий.

## Как это работает

### Основной флоу

Пользователь вводит артикул Wildberries (например, 12345678)
Система парсит товар — получает название, бренд, материал, фото с WB
Пользователь выбирает лучшие фото из найденных
Система создает эталон — красивое фото товара на белом фоне (через AI)
Из эталона генерируются сотни вариантов фото и видео в разных форматах
Все публикуется в Pinterest (сам или через креаторов)

### Главная суперсила — сеть креаторов

Вместо того чтобы загружать контент только в свой аккаунт, пользователь может:

- **Подключить 1000+ креаторов** из сервиса
- **Они загружают этот же контент** в свои аккаунты
- **Результат**: один контент работает через 1000 аккаунтов → охват растёт в тысячи раз

## Ценность для пользователя

✅ **Экономия времени** — из одного эталона генерируется весь контент  
✅ **Масштабирование** — контент работает через сеть креаторов  
✅ **Автоматизация** — Pinterest сам публикует без ручной работы  
✅ **Дешевизна** — минимальные инвестиции в контент  
✅ **Охват** — экспоненциальный рост видимости товара
