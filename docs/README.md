# Zalivai Bot — Платформа для масштабирования контента в Pinterest

## О проекте

**Zalivai Bot** — Telegram бот для создания и массового распределения контента товаров в Pinterest через сеть креаторов.

### Основная идея

1. **Пользователь** создаёт эталон товара (фото на белом фоне)
2. **Сервис** генерирует множество вариантов фото и видео контента
3. **Наносит водяные знаки** (артикул + название товара)
4. **Создаёт CSV файл** для Pinterest (до 100 пинов за раз)
5. **Pinterest автоматически публикует** контент в ленты

### Суперсила — Сеть креаторов

- Загружай контент сам или подключи 1000+ креаторов
- Каждый креатор загружает в свой аккаунт
- Результат: один контент → множество аудиторий → экспоненциальный охват

### Коммерческая модель

- Пользователь платит за создание CSV: **1₽ за пин** (строку)
- Максимум 100 пинов за раз
- Дополнительная услуга: привлечение креаторов

## Архитектура

- **Язык**: Python (asyncio, python-telegram-bot)
- **БД**: PostgreSQL (`zalivai_db`)
- **Сервисы**: `zalivai-bot` (Telegram), `zalivai-web` (веб-вьюер)
- **Развёртывание**: systemd на Linux сервере `sku`

### Основные компоненты

| Компонент | Назначение |
|-----------|-----------|
| **Сообщения** | `prompt_templates` (БД) — хранятся как шаблоны |
| **Клавиатуры** | `handlers/keyboards/` — функции `kb_*()` |
| **Логика** | Обновление сообщений через `edit_message_caption/text()` |
| **Вотермарк** | Наложение артикула на фото/видео |
| **Pinterest CSV** | Генерация и загрузка в Pinterest |

## Ключевые переменные Pinterest

Все 17 переменных доступны во всех 22 Pinterest сообщениях:

```
{available}           — доступное количество файлов
{balance}             — баланс пользователя (₽)
{max_rows}            — макс. строк на баланс
{cost_per_row}        — цена за строку/пин (₽)
{count}               — количество строк CSV
{cost}                — стоимость операции (₽)
{affordable}          — что можно позволить на баланс
{affordable_cost}     — стоимость доступного количества (₽)
{after}               — баланс после операции (₽)
{errors_count}        — количество ошибок
{requested}           — запрошено пользователем
{articles_count}      — кол-во артикулов с файлами
{articles_list}       — список артикулов
{watermarked_photos}  — фото с артикулом (готовы)
{watermarked_videos}  — видео с артикулом (готовы)
{photos_count}        — всего фото создано
{videos_count}        — всего видео создано
```

## Быстрый старт

### Локальная разработка (MacOS)

```bash
# Подключение к БД на сервере
ssh -o RequestTTY=yes -o RemoteCommand=none sku \
  "PGPASSWORD='zalivai_pass_2024' psql -h localhost -U zalivai -d zalivai_db"
```

### Развёртывание на сервере

```bash
# После git commit + push:
ssh sku "cd /var/www/bots/Zalivai_bot && git pull && \
  systemctl restart zalivai-bot zalivai-web"
```

### Проверка логов

```bash
ssh -o RemoteCommand=none -o RequestTTY=no sku \
  "journalctl -u zalivai-bot --no-pager -n 100"
```

## Важные правила

⚠️ **НЕ хардкодить логику!**
- Все сообщения → в `prompt_templates` (БД)
- Все клавиатуры → в `handlers/keyboards/`
- Все переходы → обсудить перед реализацией

⚠️ **Все изменения через git!**
- Никаких прямых правок на сервере
- Только: commit → push → деплой

⚠️ **Логика флоу — спросить!**
- Не додумывать, 100% уточнить детали
- Особенно: переходы между экранами, условия, валидация
