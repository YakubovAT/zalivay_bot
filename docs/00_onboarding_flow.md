# Флоу Приветствия (Шаги 1а-1д)

## Описание

Новый пользователь видит последовательность из 5 экранов приветствия, которые рассказывают:
- **1а** — Что делает бот (генерирует пины для Pinterest)
- **1б** — Проблема: реклама на WB/Ozon работает как счётчик такси (платишь = едешь)
- **1в** — Решение: пример продавца, который получил трафик через Pinterest без бюджета
- **1г** — Как бот помогает: ИИ генерирует пины + готовый CSV + база креаторов
- **1д** — Результаты от реальных продавцов + призыв к действию

## Навигация

Каждый экран перерисовывается (edit_message_text):
- **1а**: кнопка "Далее →"
- **1б-1г**: кнопки "← Назад" и "Далее →"
- **1д**: кнопка "Начать работу"

После "Начать работу" → переход в профиль/меню (Шаг 2)

## Условие отображения

Пока: показывать всегда при `/start`  
Будущее: только новым пользователям (флаг `is_registered`)
