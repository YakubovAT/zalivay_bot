# Флоу Приветствия (Шаги 1а-1е)

## Описание

Новый пользователь видит последовательность из 5 экранов приветствия, которые рассказывают о проблеме отсутствия бюджета на рекламу и как Pinterest помогает получить трафик:

- **Шаг 1а: Приветствие** — Привет и проблема (есть ли время на Pinterest?)
- **Шаг 1б: Проблема** — Реклама на WB/Ozon работает как счётчик такси (платишь = едешь)
- **Шаг 1в: Решение** — Пример продавца, который получил трафик через Pinterest без бюджета
- **Шаг 1г: Как помогает бот** — ИИ генерирует пины + готовый CSV + база креаторов
- **Шаг 1е: Результаты** — Результаты других пользователей и призыв к действию

## Навигация и УХ

Каждый экран **перерисовывается** одним сообщением (используется `edit_message_media`):

| Шаг | Сообщение | Кнопки |
|-----|-----------|--------|
| 1а | msg_welcome_1a | "Далее →" |
| 1б | msg_welcome_1b | "← Назад", "Далее →" |
| 1в | msg_welcome_1c | "← Назад", "Далее →" |
| 1г | msg_welcome_1d | "← Назад", "Далее →" |
| 1е | msg_welcome_1e | "← Назад", "Начать работу" |

После "Начать работу" → переход в профиль/меню (Шаг 2)

## Технические детали

### Файлы

- **Database**: `database/schema.sql`
  - 5 шаблонов: `msg_welcome_1a`, `msg_welcome_1b`, `msg_welcome_1c`, `msg_welcome_1d`, `msg_welcome_1e`
  - sort_order: 10, 11, 12, 13, 14 соответственно

- **Клавиатуры**: `handlers/keyboards/__init__.py`
  - `kb_next()` — "Далее →"
  - `kb_back_next()` — "← Назад" + "Далее →"
  - `kb_start_work()` — "← Назад" + "Начать работу"

- **Логика флоу**: `handlers/flows/onboarding.py`
  - Состояния: `_WELCOME_1A`, `_WELCOME_1B`, `_WELCOME_1C`, `_WELCOME_1D`, `_WELCOME_1E`
  - Обработчики:
    - `cb_welcome_next()` — навигация вперед
    - `cb_welcome_back()` — навигация назад
    - `cb_welcome_start_work()` — переход в профиль

### Поток данных

```
/start
  ↓
cmd_start() → Шаг 1а (msg_welcome_1a + kb_next)
  ↓
cb_welcome_next() → Шаг 1б (msg_welcome_1b + kb_back_next)
  ↓
cb_welcome_next() → Шаг 1в (msg_welcome_1c + kb_back_next)
  ↓
cb_welcome_next() → Шаг 1г (msg_welcome_1d + kb_back_next)
  ↓
cb_welcome_next() → Шаг 1е (msg_welcome_1e + kb_start_work)
  ↓
cb_welcome_start_work() → Профиль/Меню (Шаг 2)
```

Навигация "Назад" работает в обратном направлении.

## Условие отображения

**Текущее** (постоянно): показывать всегда при `/start`  
**Планируется**: только новым пользователям (флаг `is_registered=false`)
